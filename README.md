Your Gateway to Authentic Sri Lankan Experiences

LocalLens SL is a mobile application prototype built as a university project to help travelers discover authentic, hyper-local experiences in Sri Lanka, offered directly by local hosts and small independent operators.

Instead of generic tour packages, LocalLens SL focuses on niche, cultural, and community-driven experiences such as village cooking classes, hidden hiking trails, craft workshops, wildlife spotting, and photography walks.
